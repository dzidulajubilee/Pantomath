Deep extraction
Fetches each new article's full page (not just the RSS teaser) for severity/vendor/threat-actor/IOC detection — RSS summaries are usually too short to contain real indicators. Slower on sources with many new items at once; fails silently back to teaser-only text on paywalls/timeouts. On by default.

Notify me for

Browser permission: unknown. Desktop notifications only fire while this dashboard tab is open in your browser, and require a secure connection (HTTPS, or "http://localhost") — browsers silently block the permission prompt on plain HTTP at any other address, with no dialog and no error shown. Pantomath is a web app, not a native app, so there's also no background/OS-level push when the tab is closed.

Webhooks

Alert via webhook
Send a POST request when a new item matches a keyword, a specific source, and/or a minimum severity. Works over plain HTTP — unlike browser notifications, this doesn't need HTTPS.
